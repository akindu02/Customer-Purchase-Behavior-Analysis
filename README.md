# Customer Purchase Behavior Analysis

## 📌 Project Overview
This project analyzes customer purchase behavior using Python, PostgreSQL, SQL, and Power BI.  
The objective is to understand spending patterns across cities and genders, identify high‑value customers, and present key business insights through an interactive dashboard.

This is an end‑to‑end beginner‑level data analytics project covering data preparation, database management, analysis, and visualization.

---

## 🎯 Objectives
- Analyze customer spending behavior
- Identify high‑revenue cities
- Understand gender‑based customer distribution
- Highlight high‑value customers for business decision‑making
- Create a clear and interactive dashboard for stakeholders

---

## 🛠 Tools & Technologies Used
- **Python** (Pandas) – Data cleaning and preparation  
- **PostgreSQL** – Database creation and data storage  
- **SQL** – Data analysis and aggregations  
- **Power BI** – Data visualization and dashboard creation  
- **VS Code** – Development environment  
- **Git & GitHub** – Version control and project hosting  

---

## 🧹 Data Preparation
- Raw customer data was cleaned using Python and Pandas
- Duplicate records were removed
- Gender values were standardized
- Cleaned data was exported and loaded into PostgreSQL

---

## 🗄 Database & SQL Analysis
- A PostgreSQL database (`customer_db`) was created
- Customer data imported into the `customers` table
- SQL queries used to calculate:
  - Average customer spend by city
  - Total revenue
  - Customer distribution by gender
  - Identification of high‑value customers
  - Overall business KPIs

---

## 📊 Power BI Dashboard
The Power BI dashboard provides clear visual insights, including:

- **KPIs**
  - Total Revenue
  - Average Customer Spend

- **Charts**
  - Average Customer Spend by City
  - Total Spend by City & Gender (Stacked Bar)
  - Customer Distribution by Gender (Pie Chart)
  - Area chart showing spending distribution

- **Table**
  - High‑Value Customers (total_spent > 250)

The dashboard is designed for quick understanding and business decision‑making.
